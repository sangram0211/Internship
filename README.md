# Battery Inventory Management System

## Overview
The Battery Inventory Management System is a web application designed to manage and monitor battery inventory for electric vehicle swap stations. It allows administrators to add, edit, and track batteries and generate reports efficiently.

## Features
- Admin login and authentication
- Dashboard for inventory overview
- Add and edit battery details
- Battery status reports
- Session-based access control

## Technologies Used
- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS, Jinja2  
- **Database:** MySQL / SQLite

## Requirements
- Python 3.x
- Flask and required packages
